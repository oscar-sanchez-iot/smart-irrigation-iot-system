# smart-irrigation-iot-system
End-to-end IoT smart irrigation and greenhouse monitoring system using ESP32, MQTT, AWS EC2 and Node-RED.

Smart Irrigation IoT System

Meritorious Thesis Project – Systems Engineering

An end-to-end IoT solution for greenhouse monitoring and automated irrigation control, developed using ESP32, MQTT protocol, AWS cloud infrastructure and Node-RED dashboard.

Project Overview

This project was designed to automate irrigation and environmental control for small-scale crops.
The system monitors:

Temperature

Soil Moisture

Relative Humidity

pH Levels

And automatically controls:

Irrigation system

Ventilation

Curtain mechanism

System Architecture

ESP32 (sensor reading & actuator control)

MQTT protocol for communication

Mosquitto broker deployed on AWS EC2

Node-RED dashboard for monitoring and manual control

MicroSD data logging (.csv)

Custom PCB design

Cloud Infrastructure

AWS EC2 instance

Custom AMI configuration

Secure MQTT communication

Real-time remote access

Features

✔ Automatic and manual operation modes
✔ Real-time dashboard
✔ Historical data logging
✔ CSV export for analytics
✔ Alarm system for abnormal conditions
✔ Multi-crop configuration
